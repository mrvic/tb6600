# tb6600
TB6600 with stepper motor


Start arduino
connect with usb
start console
type:
xon
cdon
s60
x3000
mx
xoff



it will:
start 6600driver
start console output
set delay between steps to 60ms
set 3000 steps
move to preset step position
stop 6600 driver

For faster motor set to 1/4 microstepping: 
SW1 ON
SW2 OFF
SW3 OFF

For precize set 32 microstepping
SW1 OFF
SW2 OFF
SW3 OFF
